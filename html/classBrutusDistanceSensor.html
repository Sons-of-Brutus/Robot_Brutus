Detailed Description

Ultrasonic distance sensor handler.

Manages distance measurements using a trigger pin (TRIG) and an echo pin (ECHO). The echo pulse duration is measured via an interrupt service routine (ISR) and synchronized with a FreeRTOS semaphore.

float BrutusDistanceSensor::read_distance ( )
inline

Measure the distance to an object.

Sends a trigger pulse and waits for the echo response. If the echo is received within the timeout, the distance is calculated based on the speed of sound.

Returns
Distance in centimeters, or -1.0 if the timeout expires.
