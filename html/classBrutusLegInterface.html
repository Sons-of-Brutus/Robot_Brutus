Detailed Description

Interface class to control a single Brutus robot leg.

This class abstracts the control of a leg composed of two servos: a shoulder servo and an elbow servo, both driven by a PCA9685 controller. It provides methods to configure the servos, read the current leg state, and command a desired leg state, optionally handling angle inversion.
