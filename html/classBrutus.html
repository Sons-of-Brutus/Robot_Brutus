Detailed Description

Main control/interface class for the Brutus quadruped robot.

This class manages:

  • Four leg interfaces
  • FreeRTOS motion task for motion control (pose and speed based)
  • RGB eye status indicators

Brutus supports both direct pose control and speed-based gait generation.

Initializes the perception sensors hardware.

This method configures the GPIO pins used by the distance sensors (e.g. ultrasonic sensors) located on the right, left and front sides of the robot. Trigger pins are configured as outputs and echo pins as inputs.

After calling this method, the perception subsystem is configured at the hardware level but not yet activated.

Parameters
right_trig_pinGPIO pin connected to the right sensor trigger.
right_echo_pinGPIO pin connected to the right sensor echo.
left_trig_pinGPIO pin connected to the left sensor trigger.
left_echo_pinGPIO pin connected to the left sensor echo.
front_trig_pinGPIO pin connected to the front sensor trigger.
front_echo_pinGPIO pin connected to the front sensor echo.
